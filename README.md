# Emby全能助手 (Emby to JAVDB & External Players)

一款为 Emby 打造的全能增强脚本，集成 JAVDB 跳转与全套外部播放器。

## 主要功能

- **JAVDB 跳转**：在 Emby 影片详情页和列表页自动识别番号并添加跳转 JAVDB 的按钮。
- **外部播放器集成**：集成 15 款主流外部播放器（PotPlayer, VLC, IINA, nPlayer, MXPlayer, Infuse, MPV 等）。
- **性能优化**：移除冗余功能，优化代码逻辑，确保流畅体验。

## 安装说明

1. 安装浏览器插件 [Tampermonkey](https://www.tampermonkey.net/)。
2. 点击 [安装脚本](https://raw.githubusercontent.com/86168057/emby-to-javdb/master/emby-to-javdb.user.js)。
